# Code zur Präsentation über Black-Scholes Gleichung
Das Programm prices.py stellt Funktionen zur Berechnung des Optionswerts mithilfe der Lösungen der Black-Scholes Gleichung bereit.
Der Python-Code (main.py) nutzt die Black-Scholes Gleichung, um aus einem Optionspreis die Wahrscheinlichkeitsverteilung für Kursbewegungen zu berechnen.
